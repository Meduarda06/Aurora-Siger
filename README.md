Sistema automatizado de verificação pré-lançamento da nave **Aurora Siger** com telemetria, algoritmos e análise energética.

## 📋 Explicação do Projeto

**Objetivo:** Consolidar verificações para autorizar/abortar decolagem.

**Funcionalidades:**
- 📊 Telemetria 6 parâmetros (temp, energia, pressão)
- 🤖 Algoritmo decisão (10 verificações críticas)  
- ⚡ Análise energética (autonomia 66.7%)
- 📈 Dashboard 4 gráficos profissionais
- 💾 Export Excel/CSV automático

**Resultado:** `🚀 PRONTO PARA DECOLAR` (10/10 OK)
